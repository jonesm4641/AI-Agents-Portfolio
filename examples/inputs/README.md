# Example Inputs

This directory contains representative example inputs used to demonstrate how agents in this repository are intended to be used.

Inputs are intentionally sanitized, simplified, and non-sensitive.

---

## Input Categories

Examples may include:
- Job descriptions and resume excerpts
- Scripture references or study topics
- Static malware tool outputs (non-malicious)
- Small CSV datasets or tables
- IT service issue descriptions

---

## Disclaimer

All example inputs:
- Contain no real personal data
- Contain no live malware or exploits
- Are for demonstration and documentation purposes only
